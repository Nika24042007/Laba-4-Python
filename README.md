## Инструкция использования
### Запуск
```python -m src.main```

### Основная инструкция (алгоритм решения)
    1. Пользователь вводит номер действия из предложеных
    2. Если выбранана симуляция следует просьбы вабрать кол-во шагов и сид (обязательно должны быть целыми неотрицательными числами). Эти параметры могут отсутствовать в таком случае step = 1, seed = 0. Если сид не установлен, то программа работает без него
    3. Если выбран ручной ввод, то пользователю предложено выбрать один из вариантов действий (add, del, get, find, change, add_random, print)
    4. Далее работа зависит от выбраного действия:
        * ADD
         - Просят ввести ("название" "автор" "жанр" год isbn), год и isbn должны быть целыми неотрицательными числами
         - Если все корректно выведеся сообщение о добавление книги, если что-то пошло не так выведется сообщение об ошибке
        * ADD_RANDOM
         - Просят ввести целое неотрицательное число, количество книг, которое будет сгенерировано и добавлено в список книг
         - Сообщение об окончание заполнения списка сгенерироваными книгами
        * PRINT
         - Выводится список всех книг в библиотеке
        * DEL
         - Просят ввести индекс книги для удаления
         - Если все хорошо выведется книга, которую удалили и сообщение об  успешном удаление, если что-то пошло не так, то выведеться сообщение об ошибке или о пустоте списка
        * GET
         - Просят ввести (Название, автор), через запятую
         - Если все хорошо программа выведет найденую книги, если что-то не пошло не так выведеться сообщение об ошибке или пустоте списка или не нахождение сочетание автор-название
        * FIND
         - Просят ввести аргумент, по которому будет производиться поиск (можно ввести только один аргумент из списка) (author, title, genre, isbn, year)
         - Потом просят ввести значение аргумента для поиска
         - Если книга была найдена, то она выведется, если нет, то выведется соодщение об ошибке или пустоте списка, или отсутствие данного знасения у аргументов в списке
        * CHANGE
         - Просят ввести индкс книги, аргумент и новое значение аргумента через запятую (1, title, "Красная шапочка"). Если аргумент title/genre/author, то новое значение вводиться в кавычках, иначе без просто целое неотрицательное число
         - Потом выведется сообщение о ворректной замене или ошибке или пустоте списка
    5. Если выбирается выход то программа завершает работу и список книг без возвратно обнуляется

## Синтаксис
### Воок
    * Создается книги с параметрами задаными пользователем
    * Если используется **random_book** создается книга с рандомными значениями

### BookCollection (Наследует UserList)
    * Создается пустой список объектов класса Воок
    * **getiem_format** задается формат вывода среза списка
    * **add_random_book** добавление рандомно сгенерированой книги
    * **__str__** задает формат вывода списка при печате в консоль

### BookIndex (Наследует UserDict)
    * Создает пустой словарь
    * **build_index** Создает словарь из списка с объектами класса Book, по заране заданому параметру ключа
    * **update_index** Обновлене содержимиго словаря
    * **__str__** Формат вывода словаря при печате в консоль

### Liblary
    * Создание списка и словарей
    * **updata_indexs** Обновление значений словарей
    * **add_random** Добавление рандомно сгенерированой книги в список и обновление словарей
    * **add** Добаваление книги со значениями задаными пользователем в список и обновление словарей
    * **del_book** Удаление книги из списка по индексу заданому поьзователем, обновление словарей
    * **del_random** Удаление книги из списка по рандоному индексу, обновление словарей
    * **find_del_author_title** получение книги с определеным название и автором, если такая книга есть она выводиться в консоль, удаляется из списка и значения словаря обновляются
    * **change** замена значения параметра выбраного пользователем на новый также заданный пользователем у книги по индексу заданному пользователем, обновление словарей
    * **change_random** замена значание рандомного прараметра на новое рандомно выбраное значение, выбор книги производится по рандомно выбраному индексу, в конце происходит обновление словарей
    * **find_author** Поиск книги по автору
    * **find_isbn** Поиск книги по isbn
    * **find_year** Поиск книги по году выпуска
    * **find_title** Поиск книги по названию
    * **find_genre** Поиск книги по жанру
    * **__str__** Фоормат вывода при печати в консоль

### Simulation
    Симуляция действий в библиотеке по заддоному пользователем количеству шагов и сиду. Рузельтат каждого действия выводиться в консоль
